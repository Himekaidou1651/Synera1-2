# 古明地 こいし（Koishi）

| 属性 | 值 |
|------|-----|
| 种族 | 妖（Youkai） |
| 职业 | 魔法使（Mahoushi） |
| 生命 | 350 |
| 攻击 | 36 |
| 最大法力 | 75 |
| 普攻 | 心符「无意识弹幕」 |
| 大招 | 本能「イドの解放」 |
| 装备 | 恋之瞳 |

## 攻击范围

- **普攻**：周围 8 格所有敌人
- **大招**：曼哈顿距离 ≤4 的所有敌人 + 上方 6 格直线

## 技能说明

- **心符「无意识弹幕」**：无意识地释放弹幕，攻击周围 8 格所有敌人
- **本能「イドの解放」**：解放封闭的本我，对范围内最近目标造成伤害并溅射其周围 1 格（基础伤害 90），大范围无死角符卡
