# 博麗 霊夢（Reimu）

| 属性 | 值 |
|------|-----|
| 种族 | 人（Ningen） |
| 职业 | 巫女（Miko） |
| 生命 | 400 |
| 攻击 | 36 |
| 最大法力 | 75 |
| 普攻 | 宝符「陰陽宝玉」 |
| 大招 | 霊符「夢想封印」 |
| 装备 | 阴阳玉 |

## 攻击范围

- **普攻**：十字方向 2 格内所有敌人
- **大招**：周围 8 格所有敌人

## 技能说明

- **宝符「陰陽宝玉」**：向十字方向投掷阴阳玉，攻击范围内所有敌人，每次普攻回复 25 法力
- **霊符「夢想封印」**：对周围 8 格内最多 4 名敌人造成大量伤害（基础伤害 90），强力 AOE 清场符卡
