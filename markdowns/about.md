# Synera: Synergy Auto-Arena

20260701

---

## 简介

Synera 是一个**自走棋风格**的策略游戏，基于 **Qt 6.11.1** 和 **C++17** 开发。

在 8×8 棋盘上部署东方 Project 主题的单位，通过自动战斗、商店招募、羁绊搭配、装备收集与升星合成为核心玩法，策略性地击败每一轮的敌方阵容。

---

## 核心特色

- **9 名可选角色**：博丽灵梦、雾雨魔理沙、古明地恋、魂魄妖梦、爱丽丝·玛格特罗伊德、射命丸文、姬海棠果、十六夜咲夜、东风谷早苗
- **6 种羁绊**：人 / 妖 两大种族 + 巫女 / 魔法使 / 女仆 / 记者 四大职业，激活全局属性光环
- **升星系统**：3 合 1 升星，2★ 属性 ×1.8，3★ 属性 ×3.5
- **装备系统**：20 件东方主题装备（1★×4 + 2★×12 + 3★×4），4 槽位穿戴，支持合成与特效（暴击/吸血/护甲）
- **存档功能**：完整序列化棋盘、备战区、装备表等全部状态
- **经济系统**：利息机制 + 连胜/连败奖励 + 商店刷新
- **战斗系统**：BFS/A\* 寻路、有限状态机、多策略索敌、9 角色专属符卡技能
