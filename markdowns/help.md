## 📖 游戏总体介绍

Synera 是一个 **8×8 棋盘**上的自走棋策略游戏，基于 Qt 6.11.1 + C++17 开发。你在准备阶段从商店招募单位、购买装备、布置阵容；战斗阶段由 AI 自动操控单位 BFS/A\* 寻路、攻击并释放符卡大招。

### 三阶段循环

```
准备阶段（布阵/购物/升级人口）
    → 战斗阶段（AI 自动战斗，分步动画推进）
        → 结算阶段（奖惩/掉落/存档）→ 下一轮
```

### 核心系统速览

| 系统 | 说明 |
|------|------|
| **战斗** | 敌我交替行动（敌→我→敌→我），A\* 寻路到攻击位，被夹击时触发绝境攻击 |
| **种族** | 人（Ningen）和妖（Youkai），≥2/≥4 人分别激活 ATK+10%/+20%，≥2/≥4 妖分别激活 HP+10%/+20% |
| **职业** | 巫女（MaxMana−25）、魔法使（ATK+16%）、女仆（HP+12%）、记者（ATK+24%），≥2 同职业激活 |
| **升星** | 3 个同名同星级 → 1 个高星；倍率 1★=×1.0、2★=×1.8、3★=×3.5 |
| **装备** | 20 件模板，4 槽位，3 同类型同星可合成；可拖拽装备到单位上，信息面板可卸下；死亡掉落 1★35%/2★25%/3★15% |
| **经济** | 初始 96 金币，利息保底 5 金/回合，刷新 1 金，购买单位 2 金，购买装备按星级定价，人口升级 5 金 |
| **存档** | 完整序列化棋盘/备战区/装备表/双方状态，JSON 格式，try-catch 防护 |

### 胜负条件

玩家血量归零 → 游戏失败；敌方血量归零 → 玩家胜利。敌方单位存活时跨回合保留，与新部署单位一同再战。
